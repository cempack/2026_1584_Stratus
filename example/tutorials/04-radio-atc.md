# Radio ATC

## But

Choisir un aéroport et écouter un flux audio.

## Étapes

1. Ouvrir le répertoire ATC.
2. Chercher un aéroport.
3. Choisir un canal.
4. Cliquer sur lecture.

## Résultat attendu

- la liste d'aéroports s'ouvre,
- le flux sélectionné devient actif,
- le lecteur audio passe en lecture si la source répond.
