# ![Code](../../docs/assets/code.svg) Fiche avion

## ![Plane](../../docs/assets/plane.svg) But

Lire les détails d'un appareil sélectionné.

## ![Pencil](../../docs/assets/pencil.svg) Étapes

1. Sélectionner un avion.
2. Ouvrir ou laisser s'ouvrir la fiche.
3. Vérifier la trajectoire, la météo et la photo.

## ![Check](../../docs/assets/check.svg) Résultat attendu

- les champs de base sont remplis,
- la météo locale s'affiche,
- la trajectoire se dessine,
- une photo est chargée si la source répond.
